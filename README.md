# raytracing
A practice raytracing project in C

[![C/C++ CI](https://github.com/nahkh/raytracing/actions/workflows/c-cpp.yml/badge.svg)](https://github.com/nahkh/raytracing/actions/workflows/c-cpp.yml)

## Overview

This project is a simple raytracing engine I'm writing in C. I'm using raytracing because it's a domain I have some familiarity with, and is computationally demanding. I'm doing it in C in order to learn C :)

I'm not currently accepting submissions, as this is just a practice project for me.

I'm also not providing a license at this time, as I don't have the inclination to spend time choosing an appropriate one.


## Dependencies

* SDL2
* Posix threads (currently, will likely switch to SDL threads soon)
* check for unit testing

There are some .vscode files in the repository, but they aren't required to build or run the project.


## Setup

```
sudo apt-get install SDL2 check
make run
```
